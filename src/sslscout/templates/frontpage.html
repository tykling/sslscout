sslscout

The complexity of configuring an SSL webserver correctly has lead to the creation of different online SSL configuration checking websites that help administrators get the configuration right.

The problem with this approach is that few administrators remember to re-check their configuration periodically. While best practices concerning SSL configuration change all the time, the actual configuration of webservers is often static for years.

With sslscout administrators can configure automatic periodic checks of their websites with the different SSL-configuration checking services, and get email notifications when the site rating changes.
